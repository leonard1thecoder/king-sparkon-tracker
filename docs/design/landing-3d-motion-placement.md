# Landing page 3D motion placement

The landing page uses motion only where it explains a product idea:

- Hero: animated scan terminal and rotating 3D brand cards.
- Sponsorship: floating support visual for hosting, security and platform maintenance.
- Roles: animated access-form visual before the role cards.
- Capacity: layered administrator and owner dashboards.
- Engineering: layered feedback and product-response evidence.
- Contact: animated contact workflow above the real form.

## Visual constraints

- All resting surfaces remain white.
- Structural borders and glow use the sky-blue design tokens.
- Orange remains reserved for interactive hover feedback.
- Motion respects `prefers-reduced-motion`.
- Pricing remains focused and does not contain decorative 3D elements.
